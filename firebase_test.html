# Firebase Beginner's Guide - Step by Step ## üî• What is Firebase? Firebase is Google's platform for building web and mobile applications. It provides: - **Authentication** - User login/signup system - **Firestore** - NoSQL cloud database - **Hosting** - Web hosting service - **Storage** - File storage - And many other services ## üöÄ Step 1: Create Firebase Project ### 1.1 Go to Firebase Console - Open [https://console.firebase.google.com](https://console.firebase.google.com) - Sign in with your Google account ### 1.2 Create New Project 1. Click **"Add project"** or **"Create project"** 2. Enter project name: `MEGA-LEAN-Assessment` 3. Accept terms and click **"Continue"** 4. Choose whether to enable Google Analytics (recommended for tracking) 5. Click **"Create project"** ### 1.3 Project Dashboard You'll see your project dashboard with various services. ## üîê Step 2: Enable Authentication ### 2.1 Go to Authentication 1. In the left sidebar, click **"Authentication"** 2. Click **"Get started"** ### 2.2 Configure Sign-in Method 1. Under **"Sign-in method"** tab 2. Click **"Email/Password"** 3. Enable it (toggle switch) 4. Click **"Save"** ### 2.3 Authentication Settings - Keep default settings for now - You can add email verification later ## üóÑÔ∏è Step 3: Setup Firestore Database ### 3.1 Create Database 1. In left sidebar, click **"Firestore Database"** 2. Click **"Create database"** 3. Choose **"Start in test mode"** (allows read/write for 30 days) 4. Select a location (choose closest to your users) 5. Click **"Create"** ### 3.2 Database Structure Firestore organizes data in: - **Collections** (like tables) - **Documents** (like rows) - **Fields** (like columns) Your app will use: - `profiles` collection (user information) - `assessments` collection (assessment data) ## ‚öôÔ∏è Step 4: Get Firebase Configuration ### 4.1 Add Web App 1. Click **Project Settings** (gear icon) 2. Under **"Your apps"**, click **"Web app"** ( icon) 3. Enter app name: `MEGA Assessment Tool` 4. Click **"Register app"** ### 4.2 Copy Configuration You'll see configuration like this: ```javascript const firebaseConfig = { apiKey: "AIzaSy...your-key-here...", authDomain: "your-project-id.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project-id.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef" }; ``` **Copy this entire configuration object - you'll need it!** ## üìù Step 5: Update Your Configuration ### 5.1 Edit config_firebase.js 1.
