### **2. Taking Your First Assessment**

#### **Step-by-Step Process:**

**A. Start Assessment**
1. Click **"New Assessment"** in the navigation menu
2. You'll see the assessment form

**B. Fill in Company Details**
```
Company Name: Your organization name
Assessor Name: Your name or team name
Assessment Date: Today's date (auto-filled)
```

**C. Answer Questions**
- The assessment has **35 questions** across **7 dimensions**
- Each dimension has **5 questions**
- Rate each statement from 1 to 5:
  - **1** = Strongly Disagree
  - **2** = Disagree
  - **3** = Neutral
  - **4** = Agree
  - **5** = Strongly Agree

**D. Save Options**
- **Save Draft**: Save your progress and come back later
- **Submit Assessment**: Complete and calculate results
- **Reset Form**: Clear all answers and start over

**Tips:**
âœ… Be honest in your responses  
âœ… Consider actual practices, not aspirations  
âœ… Consult with team members if unsure  
âœ… Save drafts if you need more time  
âœ… Complete all 35 questions for accurate results

**View User Information:**

1. Find active user

**Remove Admin Rights:**
1. Find user with "Admin" role
2. Click **"Remove Admin"** button
3. User becomes regular user
4. Loses access to admin features

## âš™ï¸ **Troubleshooting**

### **Users**

**Problem: Can't sign in**
- Check email/password spelling
- Use "Forgot Password" to reset
- Ensure account is activated (contact admin)

**Problem: Lost draft assessment**
- Check "My Assessments" page
- Look for items marked "Draft"
- Click "Edit" to continue

**Problem: PDF won't download**
- Check browser popup blockers
- Try different browser
- Use Preview first to verify data

**Problem: Don't see Admin Panel**
- You may not have admin role
- Contact your administrator
- Only admins see this button

### **Administrators**

**Problem: Can't see all assessments**
- Check if filters are applied
- Clear filter boxes
- Select "All" in status dropdown

**Problem: Can't deactivate a user**
- You cannot deactivate yourself
- Check if user is already inactive
- Refresh page and try again

**Problem: CSV won't open**
- Try opening in Excel or Google Sheets
- Check file downloaded completely
- Ensure CSV viewer is installed
